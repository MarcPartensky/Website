About Me

Learning everyday because life is too short to waste any.

Royan - May 2020

My name is Marc Partensky.

I am 20 years old and I study at ISEP.

  • Actively looking for a job.
  • Hyperactive on Github.
  • 5 certificates in Deep learning
  • Fluent in English

I am used to spending most of my time coding on my multiple projects. I started coding at the age of 16 on my calculator. Then at the age of 18 I started coding with Python which quickly became my favorite programming language. I began by playing with pygame and even later made my own library on top of it named pygame-geometry which was used for most of my pygame games and also used for my fourier-drawing library in python. As soons as i learned python I started following tutorials of C, C++, Java, C#, Swift and I started to get familiar with web development as I wanted to make multiplayer games. Thus, as I entered in higher education, I learned how most of the web is set up and how http works and I learned html, css, javascript, php, sass, bootstrap, nodejs, reactjs, vuejs and django with python (this website uses django). With my newly acquired web skills I started making this website and added few of my games. For my TIPE of the first year i made a replica of the game Othello also called Reversi which can be found here. I also experimented with making games with unity and made an opengl simulation in c++ of granular flow for my TIPE in second year. Then as I always wanted to get seriously into machine learning I followed 5 lessons on Coursera on the matter and i received the certifications that come with it visible on my linkedin page. Meanwhile I started to develop a discord bot for fun to use in my own server. Now i plan to use my deep learning knowledge in order to make a it a clever chatbot which you can talk with.
